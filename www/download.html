Chess clock for Maemo 5 (N900) is available in the extras repository.  Application is Chess Clock and group games. Direct link to the package page in maemo.org downloads.

Chess clock for 'Meego 1.2 Harmattan' (N950 and N9) is available from Nokia Store.
It is also is available from a personal repository. (New versions should appear there sooner than on Nokia Store, but are not guaranteed to be stable.)
To take the repo in use create a file helihyv.lists in the /etc/apt/sources.list.d/ with the contents:
deb http://repo.pub.meego.com/home:/helihyv/Harmattan/ ./
and to install chessclock from it run the commands:
 apt-get update
apt-get install chessclock

Source code and development snapshot are available in the garage git reposity. The N900 version is on the master branch and the N9/N950 version is on the harmattan branch.

The experimental version for N800/N810 (Maemo 4.1) is available from the diablo extras-devel repository.
